# 文生视频模板（中文输出）

首轮输出 **三条** 完整提示词时：每条标题须含 **适用于：Seedance** 或 **适用于：Veo**（见 `SKILL.md`）。让 **A/B/C 在镜头路线或节奏上实质不同**（例如 A 双镜头切换、B 一镜跟拍、C 固定镜头重表演），同时保持同一故事内核与用户意图。修订与跨模型适配同理在标题中标明适用模型。

本文件用于视频任务改写，支持：
- Veo 系列
- Seedance 系列

## A. Seedance 模板

适用：多镜头快节奏、短时叙事、情绪推进明确的片段。

与火山 Seedance 类官方提示规范对齐的 **书写顺序**（成文时可仍为一段中文，但逻辑分段要清晰）：

1) **片段目标**（一句话）  
2) **主体**（谁/什么）  
3) **主体动作 / 表演**（演员在做什么——**单独表述**，勿与运镜揉成一团）  
4) **环境 / 场景**  
5) **镜头与运镜**（景别、机位、推拉摇移；**本条只保留一个主干运动**，需要第二条运动时改用时间切片或「随后」明确切换，避免两句冲突指令并存）  
6) **按时间的节拍**（可选：第 N 秒发生什么）  
7) **光线与声音**（对白见下）  
8) **风格与整体节奏**  
9) **约束 / 不希望出现的元素**  

**对白**：下游文档常见写法为用 **双引号** 包住台词，便于口型对齐。

**参考素材占位符**（与接入平台一致，勿混用两套）：  
- 国内 **即梦** 等界面：常用 `@图片1`、`@视频1`、`@音频1`  
- **英文 API README**（如 Replicate Seedance）：常用 `[Image1]`、`[Video1]`、`[Audio1]`  

用户未说明平台时，优先与其产品界面一致；不确定时可选用其一并在对话里约定。

参考骨架：

```txt
请生成一段[时长]秒的[风格]视频。主体：[人物/物体]。主体动作：[表演节拍，可与时间轴对应]。环境：[场景描写]。镜头（单独一句）：从[起始景别/机位]出发，以[一种主运动：推/拉/摇/移/跟…]为主；若需切换镜头，写「随后切至…」避免互相矛盾。光线：[…]。声音：环境声[…]；若有台词：「[对白原文]」。节奏与情绪：[…]。避免出现[不需要元素列表]。
```

参考素材模式附加句（二选一，与平台一致）：

```txt
使用 @图片1 作为角色外观参考，@视频1 作为动作或运镜参考，@音频1 作为节奏或 BGM 参考。
```

```txt
使用 [Image1] 作为角色外观参考，[Video1] 作为动作或运镜参考，[Audio1] 作为节奏或 BGM 参考。
```

## B. Veo 模板

适用：强调对白口型、音画同步、商业质感、声音设计完整的场景。

与 Vertex「Anatomy of a Veo prompt」及 DeepMind Veo 指南对齐的输出结构：

1) **镜头 / 叙事目标**（一句话）  
2) **主体**  
3) **动作**  
4) **场景 / 语境**（短片默认 **单一场景**，除非用户明确要求多场景）  
5) **镜头角度与运动**（可与焦点变化一并写）  
6) **风格与氛围**（含色彩与后期气质）  
7) **对白 / 旁白**（如有）  
8) **声音分层**：环境声、关键 **SFX**、BGM 气质（可按 DeepMind 示例分列描写）  
9) **约束**：Vertex 文档建议 negative 侧尽量用 **「想要的效果」** 正向表述，减少单纯「不要××」堆砌  

**图生视频**：若用户已提供首帧/参考图，提示词侧重 **运动、镜头与 temporal 变化**，避免重复描述已定静态构图。

**对白写法与平台**：  
- **Gemini / DeepMind Veo 示例流**：对白可用 **引号** 包原文，并强调口型同步。  
- **Vertex API 最佳实践** 中有慎用嵌套引号的提示：若用户声明走 Vertex，对白可改为「角色自然说出以下台词：[原文]」或类似结构，减少多层引号。

参考骨架（DeepMind / Flow 友好）：

```txt
生成一段[时长]秒、单场景视频：在[场景/语境]中，[主体]进行[动作]。镜头：[角度与运动]，焦点从[…]过渡到[…]。表演细节：[…]。对白：“[台词原文]”（口型与语音同步自然）。声音：环境声[…]；关键 SFX：[…]；背景音乐：[风格与音量关系]。画面风格与氛围：[…]。约束以正向为主：[例如「画面保持干净无遮挡字幕」]。避免出现[少量必要否定项]。
```

若有对白，附加硬约束句：

```txt
对白节奏自然，口型与语音严格同步，不要出现无声张口或口型漂移。
```

## 文生视频改写补全清单

如果用户输入过短，至少补全：
- 时长（默认 5-8 秒）
- 主体与场景
- 镜头（景别 + 运动）
- 动作节拍（时间切片）
- 光线与声音
- 风格与情绪弧线
- 禁止项

若用户提供素材，再补：
- 引用角色分配（谁做外观参考，谁做动作参考，谁做声音参考）；占位符与平台一致（`@图片1` 或 `[Image1]` 等，勿混用）
